# Flexiwind CLI Documentation

Welcome to the Flexiwind CLI docs. Use the links below to explore commands and usage.

## Getting Started

Install globally:

```bash
composer global require unoforge/flexi-cli
```

Run help:

```bash
flexi-cli --help
```
